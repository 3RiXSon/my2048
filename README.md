# my2048

A simple iOS game where players swipe to merge tiles in a 5x5 grid. The goal is to combine blocks with same numbers and create the highest possible value. Inspired by the classic "2048" game.

## Features To Be Implemented:
- 5x5 grid mechanic
- Swiping to move tiles
- Block merging with numbers doubling
- Random block spawning (labeled `2` or `4`)

Stay tuned for updates!
